How it works:

1. Adjust the offset (top, left) and dimensions (width, height) using the controls above
2. Click "Show Preview Area" to see where the highlight will appear (green box)
3. Click "Start Tour" to see the actual tour with overlay and tooltip
4. The tooltip will appear on the right side of the highlighted area

Example usage in code:
{
  offset: { top: 250, left: 100 },
  width: 400,
  height: 200,
  title: "Your Title",
  description: "Your Description"
}
