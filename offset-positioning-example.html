GuideChimp - Offset Positioning Examples

This example demonstrates the three positioning modes:
1. Offset-based - Position tooltip using offset coordinates (no element)
2. Element-based - Position tooltip relative to an element
3. Centered - Center tooltip when neither element nor offset is provided

This is a demo element for element-based positioning

Code Examples

1. Offset-Based Positioning (No Element)

// Position tooltip at specific coordinates
const tour = [
    {
        offset: { top: 100, left: 50 },
        title: 'Top-Left Position',
        description: 'Tooltip positioned at top: 100px, left: 50px'
    },
    {
        offset: { bottom: 20, right: 20 },
        title: 'Bottom-Right Position',
        description: 'Tooltip positioned at bottom: 20px, right: 20px'
    },
    {
        offset: { top: '50%', left: '50%', transform: 'translate(-50%, -50%)' },
        title: 'Custom Center',
        description: 'Using offset with transform for centering'
    }
];

const guide = new GuideChimp(tour);
guide.start();

2. Element-Based Positioning

// Position tooltip relative to an element
const tour = [
    {
        element: '#demo-element',
        position: 'bottom',
        title: 'Element Tooltip',
        description: 'This tooltip is positioned relative to the element'
    },
    {
        element: '#demo-element',
        position: 'right',
        title: 'Right Position',
        description: 'Positioned to the right of the element'
    }
];

const guide = new GuideChimp(tour);
guide.start();

3. Centered Positioning (No Element, No Offset)

// Center tooltip when neither element nor offset is provided
const tour = [
    {
        title: 'Welcome!',
        description: 'This tooltip is centered on the screen'
    },
    {
        title: 'Another Message',
        description: 'Still centered as no element or offset is specified'
    }
];

const guide = new GuideChimp(tour);
guide.start();

4. Mixed Tour (Combining All Modes)

// Mix different positioning modes in one tour
const tour = [
    {
        title: 'Step 1: Centered',
        description: 'No element or offset - centered by default'
    },
    {
        element: '#demo-element',
        position: 'bottom',
        title: 'Step 2: Element-based',
        description: 'Positioned relative to an element'
    },
    {
        offset: { top: 50, right: 50 },
        title: 'Step 3: Offset-based',
        description: 'Positioned at specific coordinates'
    }
];

const guide = new GuideChimp(tour);
guide.start();
